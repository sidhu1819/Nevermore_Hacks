Project Lifecycle

Day 1 • Morning

Phase 1: Blueprint

  • Problem statement clean and clear description
  • Solution ideas and final idea selection
  • UML diagrams of solution
Day 1 • Afternoon

Phase 2: Data Foundations

  • Data collection (Public Datasets, Live API pulling)
  • Warning: Do not use synthetic data
  • Data Preprocessing & EDA
  • Data Visualizations
Day 2 • Morning

Phase 3: Intelligence

  • Advanced preprocessing
  • ML Model Development
  • Fine-tuning & Hyperparameter optimization
Day 2 • Afternoon

Phase 4: Deployment

  • Create GitHub Repository
  • Deployment (Streamlit, Vercel, Render, etc.)
  • Generate Live Server Link
Day 3 • Finals

Phase 5: The Review

(Only for shortlisted teams)

  • Present project to review panel
  • Presentation must contain PPT
  • Demonstrate Live Web App / Native Application
